## 專案背景

我想把 AI 工具用在真正能累積的地方，不只是臨時問答，而是讓它協助我整理資料、修改網站、建立 project pages、產生貼文草稿和管理學習流程。

## 系統想要做到什麼

- 用 GitHub 管理網站內容和版本
- 用 Markdown 新增貼文與專案
- 用 Codex 協助修改網站架構
- 用 ChatGPT 整理想法、草稿和學習計畫

## 和貼文的差別

貼文比較像一次性的想法紀錄；這個 project page 會記錄整個系統如何被建立、如何變好，以及哪些流程真的對我有幫助。
